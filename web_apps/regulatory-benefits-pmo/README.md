# Regulatory Benefits PMO Studio

This subsite is a first integrated environment for a programme office in a regulated setting where programme benefits are sensitive to variation.

## Pull-forward apps

- `Project_Health_Atlas` becomes the front-door benefit and constraint atlas.
- `regulatory-negotiation-rehearsal-board` supplies consent thresholds, pivotal parties, levers, clause families, and receipts.
- `governance-trio-replay-rules-lineage` supplies deterministic replay, rule receipts, and lineage/provenance patterns.
- `scenario_barrier_proof_fca_lattice` supplies scenario-barrier-proof gap logic.
- `portfolio_loom` supplies temporal benefit sensitivity and scenario fibres.
- `counterfactual_programme_steering` supplies causal intervention framing.
- `intent_field_navigator` supplies CSV-first benefit intent alignment.
- `interface_maturity_simulator` supplies coupling and spillover logic for programme interfaces.
